# AMV-Converter
A simple AMV converter.

Steps:

1. Download ZIP and extract it into your favorite IDE (eg. VS code).
2. Add a folder called "uploads" (case sensitive).
3. Then save it.
4. To start it run "npm start" and it should output "
> amv-convert@1.0.0 start
> node server.js

Server running on port 3000"
5. Then open localhost:3000 and use it!
